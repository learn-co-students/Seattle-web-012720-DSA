# Purge and Organize

- Given an array of integers, return an array which contains no duplicates and is sorted

## Examples

```javascript
uniqueSort([1, 2, 4, 3]) 
    ➞ [1, 2, 3, 4]

uniqueSort([1, 4, 4, 4, 4, 4, 3, 2, 1, 2]) 
    ➞ [1, 2, 3, 4]

uniqueSort([6, 7, 3, 2, 1]) 
    ➞ [1, 2, 3, 6, 7]
```
